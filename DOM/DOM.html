El DOM (Document Object Model) es una interfaz de programación para los documentos web. Proporciona una representación estructurada del documento HTML o XML y define cómo los programas pueden interactuar con la estructura del documento, cambiar su contenido, estructura y estilo visual. El DOM representa el documento como un árbol de nodos, donde cada nodo representa una parte del documento, por ejemplo, un elemento HTML, un atributo, o incluso texto. Nodos En el DOM, un nodo es la unidad básica de la estructura de un documento. Un documento se compone de varios nodos, que pueden ser de diferentes tipos: Elemento: Representa cada etiqueta HTML en el documento. Por ejemplo,

Objetos En el contexto del DOM, cada nodo es también un objeto. En JavaScript, estos objetos pertenecen a diferentes clases según el tipo de nodo: HTMLElement para elementos HTML, Text para nodos de texto, Attr para atributos, y así sucesivamente. Cada uno de estos objetos tiene propiedades y métodos que permiten manipular sus características y comportamientos. Ejemplo de manipulación de objetos DOM en JavaScript Considera el mismo documento HTML anterior. Cambiando esta propiedad, modificamos el texto visible en la página. El DOM y su manipulación a través de JavaScript son fundamentales para la creación de páginas web interactivas, permitiendo a los desarrolladores modificar dinámicamente el contenido y la apariencia de las páginas.
